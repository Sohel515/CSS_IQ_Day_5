Que-26:- What are CSS combinators

Ans:- Combinators combine the selectors to provide them a useful relationship and the position of content in the document.
This comes in handy when you want to style certain elements all at once rather than styling elements individually.
CSS combinator selectors are as follows.


  1. Descendant Selector (space :- select the descendants of the first selector)
  2. Child Selector (> :- targets direct children of the selector)
  3. Adjacent Sibling Selector (+ :- targets the immediate siblings aafte the first selector )
  4. General Sibling Selector (~ :-targets all the following )



Que-27:-What are pseudo selectors in CSS

Ans:- A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s).
For example, :hover can be used to change a button's color when the user's pointer hovers over it.
 a:hover {
    background-color: voilet;
  }
 
 Syntax: 
 
 selector:pseudo-class {
     property : value;
  }

  and some  other example are:.

  input:active{
    border: none;
}

input:focus{
    border: none;
    outline: none;
}

button:disabled{
    color: red;
}


button:enabled{
    color:aqua;
}

div:empty{
    background-color: aqua;
    height: 100px;
}

a:link{
    color: blue;
